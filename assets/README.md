# Assets Directory

Esta carpeta contiene capturas de pantalla y recursos visuales del proyecto.

## Capturas Requeridas

Para completar el README, agrega las siguientes imágenes:

1. `dashboard_preview.png` - Vista general del dashboard
2. `dashboard_main.png` - Dashboard principal con métricas
3. `temporal_analysis.png` - Análisis temporal
4. `geographic_analysis.png` - Análisis geográfico

## Cómo Capturar

1. Ejecuta el dashboard: `streamlit run app/dashboard.py`
2. Toma capturas de pantalla de cada sección
3. Guárdalas en esta carpeta con los nombres indicados arriba
